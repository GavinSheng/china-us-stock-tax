# 境外股票收入个人所得税计税方案

> 适用对象：中国税收居民个人，持有境外（美/港股）上市公司股票及股权激励
>
> 生成日期：2026-05-24

---

## 一、法规依据

### 1. 财产转让所得

- 《中华人民共和国个人所得税法》第六条：财产转让所得，以转让收入减除财产原值和合理费用后的余额为应纳税所得额，适用 **20%** 税率。
- 《个人所得税法实施条例》第十四条：财产转让所得，以**一次**转让财产的收入减除财产原值和合理费用后的余额，为应纳税所得额。

### 2. 股息红利所得

- 《个人所得税法》第三条：利息、股息、红利所得，适用比例税率，税率为 **20%**。
- 境外上市公司分红，中国居民个人需在中国境内补缴差额（境外已预扣税款可按规定抵免）。

### 3. 境外税收抵免

- 《个人所得税法》第七条：居民个人从中国境外取得的所得，可以从其应纳税额中抵免已在境外缴纳的个人所得税税额，但抵免额不得超过该纳税人境外所得依照本法规定计算的应纳税额。
- 《关于境外所得有关个人所得税政策的公告》（财政部 税务总局公告 2020 年第 3 号）：采用**分国限额法**计算境外税收抵免。

### 4. 境外所得申报

- 《个人所得税法》第十条：从中国境外取得所得的居民个人，应当在取得所得的次年三月一日至六月三十日内办理纳税申报。

### 5. RSU 归属收入

- 《关于个人股票期权所得征收个人所得税问题的通知》（财税〔2005〕35 号）及后续文件：RSU 归属属于**工资薪金所得**。
- 《关于个人所得税法修改后有关优惠政策衔接问题的通知》（财税〔2018〕164 号）：居民个人取得股票期权等股权激励，在 2027 年 12 月 31 日前，不并入当年综合所得，全额单独适用综合所得税率表计算纳税。
- 实际缴纳：公司在中国境内代扣代缴，归属时已完成纳税义务。
- 注：财税〔2018〕164 号中 RSU 单独计税优惠政策有效期至 2027 年 12 月 31 日，到期后政策如未延续，RSU 归属可能需并入综合所得计税，届时需重新评估。

## 二、应税收入分类

| 收入类型 | 中国税法分类 | 税率 | 计税依据 | 是否在本工具计算 |
|----------|-------------|------|---------|----------------|
| RSU 归属 | 工资薪金所得 | 3%~45%（单独计税） | 归属日 FMV × 股数 | 是（境内公司已代扣，直接冲减应补缴） |
| 股票卖出盈利 | 财产转让所得 | 20% | 卖出价 − 成本（FIFO）− 合理费用 | 是 |
| 分红 | 股息红利所得 | 20% | 全额 | 是 |
| 利息所得（债券利息、股票借贷、股票收益计划）| 利息、股息、红利所得 | 20% | 全额 | 是 |
| 股票亏损 | — | 不征税 | — | 仅参与年度净额抵扣 |

## 三、卖出计税方式（两种方法自动择优）

### 方法 A：逐笔计算

每笔卖出独立计税，盈利 × 20%，亏损不参与抵扣。

**适用场景：** 全年无亏损，或亏损无法有效抵扣盈利。

### 方法 B：年度净额计算

年度内所有卖出盈亏相抵后，按净盈利 × 20%。

**适用场景：** 当年存在亏损卖出，且亏损金额能够抵扣部分盈利。

**法规分析：**

《实施条例》第十四条规定"每次转让"为一次，严格字面含义支持方法 A。但境外所得按年度汇算清缴申报，申报表按所得类别汇总填报，**实务中采用年度净额法被广泛接受**。

### 示例对比

| 交易 | 盈亏（CNY） | 方法 A 税额 | 方法 B |
|------|-------------|------------|--------|
| AAPL 卖 100 股 盈利 | +10,650 | 2,130 | |
| GOOGL 卖 50 股 亏损 | -7,150 | 0 | |
| **合计** | 净盈利 3,500 | **2,130** | **700** |

差额：1,430 元（年度净额法更优）

### 系统行为

- 当存在亏损且年度净额 < 逐笔计算税额时，**自动选择年度净额法**
- 当无亏损或年度净额 ≥ 逐笔计算税额时，**使用逐笔计算法**
- CLI 输出和 CSV 报表均会标注所选方法及备选方案信息

## 四、境外税收抵免（分国+分项限额法）

### 原理

依据财税〔2020〕3号，境外税收抵免应**分国且分项**计算：
- **分国**：美国、香港分别计算
- **分项**：资本利得、股息红利、利息所得分别计算

不同国家之间、不同收入类别之间的抵免限额**不得互相调剂**。

即共有 6 个独立抵免池：美国资本利得、美国分红、美国利息、香港资本利得、香港分红、香港利息。
每个池内：抵免额 = min(该池中国应纳税额, 该池境外预扣税额)

### 费用处理

交易费用在 FIFO 成本核算阶段已从盈亏中扣减（`fifo.py:141`），**不在税额层面再重复抵免**，避免双重享受。

### 超额结转

某年度超额境外预扣税可向后结转 **5 个纳税年度**。结转信息存储在 `foreign_tax_credit_carryforward` 表中。

### 示例：美股分红

| 项目 | 金额 |
|------|------|
| 分红金额（USD） | 200 |
| 汇率 USD/CNY | 7.1 |
| 分红金额（CNY） | 1,420 |
| 中国应纳税额（20%） | 284 |
| 美国预扣（10%，W-8BEN） | 20 USD = 142 CNY |
| 抵免额 | 142 |
| 应补缴 | 142 |

## 五、FIFO 成本核算

### 规则

- 每只股票维护一个 FIFO 队列（tax lot）
- 买入/RSU 归属时入队，记录取得日期、成本、股份来源
- 卖出时按 FIFO 顺序出队，逐批计算已实现盈亏
- 支持导入上年末剩余持仓（跨年成本追溯）

### RSU 归属成本追溯

RSU 归属时，成本 = 归属日 FMV（即归属价格）。后续卖出时，按 FIFO 从该批次出队，计算卖出价与归属日 FMV 的差额作为盈亏。

### 跨年持仓

年末保存剩余持仓到 JSON 文件，次年计算时导入：

```bash
python -m src.cli calc txns.csv --year 2025 --lots-file lots_2024.json --save-lots
```

## 六、期权交易税务处理

### 1. 期权买入后卖出

- 买入期权支付的权利金构成投资成本
- 卖出期权收取的权利金构成转让收入
- 差额（卖出价 − 买入价 − 合理费用）作为**财产转让所得**，适用 20% 税率
- 系统通过 FIFO 跟踪每个期权 lot 的成本

### 2. 期权行权后卖出

- 行权时以行权价买入标的股票
- 后续卖出时，以行权价（实际买入成本）作为 FIFO 成本基数
- 卖出价与行权价的差额作为财产转让所得
- 期权权利金已包含在行权后的综合成本中

### 3. 期权过期作废

- 期权到期未行权也未卖出，全部权利金作为**已实现投资损失**
- 损失金额可参与当年年度净盈亏抵扣（与股票买卖亏损合并计算）
- 系统自动识别过期期权，将其成本作为负盈亏计入年度净计算

### 4. 税务处理依据

- 期权属于金融衍生工具，其交易损益参照财产转让所得征税
- 期权买入支付的权利金视为取得金融资产的成本
- 期权过期损失视为金融投资损失，中国税法无特别规定，实务中纳入年度盈亏相抵处理

## 七、CLI 使用

### 1. 解析月结单 PDF

```bash
python -m src.cli parse input/ --broker futu -o output/futu_txns.csv
```

### 2. 计算个税

```bash
python -m src.cli calc output/futu_txns.csv \
  --year 2025 \
  --exchange-rate-file rates.csv \
  --lots-file output/tax_2024/lots_2024.json \
  --save-lots \
  -o output/tax_2025
```

### 3. 一键全流程

```bash
python -m src.cli run input/ --year 2025 --broker futu
```

### 输出

- `tax_summary.csv` — 税务汇总（含每笔明细）
- `tax_detail_capital_gain.csv` — 资本利得明细
- `tax_detail_dividend.csv` — 分红明细
- `lots_2025.json` — 年末剩余持仓（用于次年导入）

## 八、系统限制

| 限制项 | 说明 |
|--------|------|
| 不支持的交易类型 | ISO/NSO 非法定期权、ESPP 员工购股计划、配股/供股、拆股/合股 |
| 亏损结转 | 中国税法不支持资本亏损跨年度结转，当年亏损当年消化 |
| RSU 代扣税处理 | RSU 归属时的中国境内代扣个税属于国内税，**直接冲减应补缴税额**，不参与境外税收抵免（FTC）流程 |
| 境外税收抵免结转 | 超额境外预扣税可向后结转 5 年，系统自动记录和跟踪 |
