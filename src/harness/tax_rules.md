> 本文件是税务计算系统的「单一事实来源」（SSOT），所有校验规则、计税逻辑、合规要求以此为准。

## 一、计税规则

### 1.1 收入分类与税率

| 收入类型 | 中国税法分类 | 税率 | 计税依据 | 本系统处理 |
|----------|-------------|------|---------|-----------|
| 股票/期权卖出盈利 | 财产转让所得 | 20% | 卖出价 − FIFO 成本 − 合理费用 | ✅ 计算 |
| 分红 | 股息红利所得 | 20% | 全额 | ✅ 计算 |
| RSU 归属 | 工资薪金所得 | 3%~45%（单独计税） | 归属日 FMV × 股数 | ✅ 计算（境内代扣直接冲减应补缴，不参与境外抵免） |
| 期权过期 | 财产转让损失 | — | 全部成本 = 损失 | ✅ 计算（proceeds=0） |

### 1.2 FIFO 成本核算

- 每只股票维护 FIFO 队列
- 买入/RSU 归属时入队
- 卖出时按 FIFO 顺序出队
- 跨年持仓从上年度末结转

### 1.2.1 年度结转规则（Carryforward）

> **核心原则**：中国个税按年度计算，使用上一年度12月31日期末持仓作为本年度税务起始成本基础。

**合规依据**：
1. 年度核算原则：中国个税按年计算，每年以年初持仓状态为起点
2. 期末快照即成本：券商期末概览中的"成本价"或"总市值"已包含买入成本+佣金，可直接用作结转成本
3. 无转仓 = 连续性：2024年12月31日持仓 = 2025年1月1日持仓（无转仓操作时）
4. 税务审计接受：券商官方月结对账单的期末持仓是审计认可的成本基础凭证

**实现要求**：
- 结转批次 `acquisition_type = 'carryforward'`
- 结转日期 `acquisition_date = '2024-12-31'`
- `broker_code` 标注来源券商（longbridge / boci / futu）
- 成本价优先取券商月结单中的 avg_cost，无则取期末收盘价
- 期权合约结转成本 = 权利金（premium），非行权后的正股
- 不同券商的同一股票各自独立结转，FIFO 消耗按券商独立追溯

### 1.3 卖出计税方式（自动择优）

- **方法 A（逐笔）**：每笔盈利 × 20%，亏损不参与抵扣
- **方法 B（年度净额）**：全年盈亏相抵后 × 20%
- **选择逻辑**：当存在亏损且年度净额税额 < 逐笔税额 → 选年度净额
- **法律依据**：《个人所得税法实施条例》第十九条，两种方法均有法律基础
- **一致性**：同一纳税年度内方法保持一致，不同年度可独立选择

### 1.4 境外税收抵免

- 方法：分国+分项限额法（US / HK 分别计算，capital_gain / dividend / interest 分项）
- 抵免额 = min(境外预扣, 中国应纳税)
- 超额部分不可用于其他国家或收入类别的抵免
- 超额部分可向后结转 5 个纳税年度（依据财税〔2020〕3号第五条）

### 1.5 汇率

- **年度汇算清缴（首选）**：纳税年度最后一日（12月31日）汇率中间价
  - 依据：《实施条例》第三十二条，"应补缴税款部分按年末汇率折合"
- 回退：精确日期 → 7 日窗口 → 默认（USD/CNY=7.10, HKD/CNY≈0.9103）
- 汇率数据来源：PBOC 每日中间价（`output/exchange_rates.csv`）

## 二、数据质量规则（Harness 校验点）

| 规则 ID | 规则 | 严重级别 | 说明 |
|---------|------|---------|------|
| IV-001 | 交易日期必须在合理范围内 | ERROR | 不能晚于今天，不能早于 2020-01-01 |
| IV-002 | 数量必须为正整数 | ERROR | 买入/卖出/分红的 quantity > 0 |
| IV-003 | 价格必须为非负数 | ERROR | price >= 0 |
| IV-004 | 金额 = 数量 × 价格（±容差） | WARNING | 容差 1% |
